# useFullscreen

将DOM进行全屏显示或关闭全屏显示

# 代码演示


<script setup lang='ts'>
import {useFullscreen} from "@lib";
import { ref } from 'vue'

const divRef = ref<HTMLElement | null>(null);
const { setFull, exitFull, toggleFull, isFullscreen } = useFullscreen(divRef, { onExitFull: ()=> {console.log('关闭了')}, onFull: () => {console.log('打开了')} })
</script>

# 基础使用

是否全屏：false

# Api

参数说明类型默认值
setFull开启全屏() => void-
exitFull退出全屏() => void-
toggleFull切换全屏/退出全屏() => void-
isFullscreen全屏状态Ref(boolean)false

# 参数

参数说明类型默认值
target目标对象`Ref<HTMLElementnull>
options选项Object-

# options

参数说明类型默认值
onExitFull退出全屏时回调函数Function() => {}
onFull开启全屏时回调函数Function() => {}
